# Habitus

Instrucciones para reemplazar el audio:

- Coloca el nuevo archivo de audio proporcionado con nombre `silencio.mp3` en la ruta `assets/audio/silencio.mp3` dentro del repositorio.
- Si quieres que yo añada el archivo al repositorio, responde confirmando y lo incorporaré.

Nota: la página ahora carga `assets/audio/silencio.mp3` como audio silencioso usado por la aplicación.
